## Most Used Shortcuts

### For Counter Staff:
1. **F2** - Create new order instantly
2. **F9** - Print bill after order
3. **F3** - Search for customer/order
4. **Escape** - Close popups quickly

### For Managers:
1. **F4** - Quick jump to dashboard
2. **Alt+4** - View reports
3. **Alt+2** - Customer management
4. **F5** - Refresh data

## Tips for Windows Users

- Use **Ctrl** key for most shortcuts (Ctrl+N, Ctrl+F, etc.)
- **Function keys (F1-F12)** work directly without holding any modifier
- **Alt + Number** shortcuts help navigate between main sections
- **Escape** is your friend - it closes any dialog or popup

## Mobile/Touch Users

- Swipe from left edge to show/hide sidebar
- Tap the keyboard icon (⌨) in the header to see shortcuts
- All functions are also accessible via on-screen buttons
